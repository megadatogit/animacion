Descripción de la actividad.

En equipo diseñaran la interfaz de la página de inicio y la página de error 404 de un sitio web (el tema será seleccionado por el equipo). La interfaz debe contener al menos cinco iconos (por ejemplo: redes sociales, carrito de compra, contacto, botones, etc.) y estos deben ser animados con la herramienta SVGator vista en clase. A su vez, se debe desarrollar la página de error 404 y, como elemento central, debe tener un SVG animado. La animación puede ser desarrollada con SVGator. La entrega se debe realizar en formato ZIP y debe incluir toda la carpeta de desarrollo, es decir, los archivos HTML, CSS y SVG utilizados para la creación de la interfaz. Transición de figuras circulares con rotate

Se utilizan elementos como transform, rotate, translate para gener la perspectiva tridimencional. Se agrupan elementos con
< g > permitiendo trancisionar los elementos del aro por secciones.

Código CSS (fragmento)

/* Trancisiona del aro mayor */
.aroExterno {

animation: aroExterno 2s ease infinite alternate;

Gatito en bucle

Gatito en bucle es una animación SVG interactiva, diseñada para acompañar procesos de carga o agregar dinamismo visual a interfaces web. La animación cobra vida al interactuar con el cursor: al pasar el mouse sobre ella (hover), el gatito inicia una secuencia de movimiento fluido
Técnicas utilizadas: Animación de trazos con stroke-dashes. Movimiento circular mediante transformaciones rotate y curvas de aceleración suaves como ease-out y base-in-sine

Código CSS (fragmento)


#tail {
 stroke-dasharray: 188; /* Circunferencia aprox */
 stroke-dashoffset: 188;
 transition: stroke-dashoffset 1s ease-out;
}

.cat-loader:hover #tail {
 stroke-dashoffset: 0;
}

Animación 5

Pajarito animado es una animación SVG interactiva, diseñada para dar vida y personalidad a interfaces web o procesos de carga. La animación presenta un encantador pajarito que cobra movimiento dinámico: las alas baten, el pico se mueve, las patitas realizan pequeños ajustes de posición y los ojos parpadean, creando una experiencia visual cautivadora.
Técnicas utilizadas: Transformaciones rotate para el movimiento de alas y pico, propiedades scale para efectos de alas, opacity para parpadeos instantaneos y curvas de animación orgánicas como ease-in-out y cubic-bezier.
