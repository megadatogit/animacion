POR PAMELA

TIKTOK

Animación en bucleque que donde se transforma y se mueve el logo de la aplicación. Se utilizan propiedades como position, stroke offset y scale permitiendo generar la animación.

/* Animación de rotación del logo */
.logoAnimado {
transform-origin: center;
animation: girar 4s ease-in-out infinite;
}

@keyframes girar {
0% { transform: rotate(0deg); }
100% { transform: rotate(360deg); }
}
#position #stroke offset #scale

POR DIANA

GATITO

Gatito en bucle es una animación SVG interactiva, diseñada para acompañar procesos de carga o agregar dinamismo visual a interfaces web. La animación cobra vida al interactuar con el cursor: al pasar el mouse sobre ella (hover), el gatito inicia una secuencia de movimiento fluido Técnicas utilizadas: Animación de trazos con stroke-dashes. Movimiento circular mediante transformaciones rotate y curvas de aceleración suaves como ease-out y base-in-sine

#tail {
stroke-dasharray: 188; /* Circunferencia aprox */
stroke-dashoffset: 188;
transition: stroke-dashoffset 1s ease-out;
}

.cat-loader:hover #tail {
stroke-dashoffset: 0;
}
...
#stroke-dashes #mouse-hover #ease-out

POR ALFREDO

PAJARITO

Pajarito animado es una animación SVG interactiva, diseñada para dar vida y personalidad a interfaces web o procesos de carga. La animación presenta un encantador pajarito que cobra movimiento dinámico: las alas baten, el pico se mueve, las patitas realizan pequeños ajustes de posición y los ojos parpadean, creando una experiencia visual cautivadora.

.animacion-1{
display: inline !important;
fill: rgb(0, 0, 0) !important;
fill-opacity: 1 !important;
filter: none !important;
mask: none !important;
opacity: 1 !important;
stroke: none !important;
stroke-opacity: 1 !important;
stroke-width: 1px !important;
transform-origin: 0px 0px !important;
visibility: visible !important;
}
#opacity #scale #rotate
