Actividad

Descripción de la actividad. En equipo diseñaran la interfaz de la página de inicio y la página de error 404 de un sitio web (el tema será seleccionado por el equipo). La interfaz debe contener al menos cinco iconos (por ejemplo: redes sociales, carrito de compra, contacto, botones, etc.) y estos deben ser animados con la herramienta SVGator vista en clase. A su vez, se debe desarrollar la página de error 404 y, como elemento central, debe tener un SVG animado. La animación puede ser desarrollada con SVGator. La entrega se debe realizar en formato ZIP y debe incluir toda la carpeta de desarrollo, es decir, los archivos HTML, CSS y SVG utilizados para la creación de la interfaz. Animación 2

Transición de figuras circulares con rotate

Se utilizan elementos como transform, rotate, translate para gener la perspectiva tridimencional. Se agrupan elementos con
< g > permitiendo trancisionar los elementos del aro por secciones.

Código CSS (fragmento)

/* Trancisiona del aro mayor */
.aroExterno {
transform: rotateY(0deg) translateX(0px);
animation: aroExterno 2s ease infinite alternate;
}
@keyframes aroExterno {to {transform: rotateY(45deg) translateX(-80px);}}
